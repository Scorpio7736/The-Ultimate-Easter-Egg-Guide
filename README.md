# The Ultimate Easter Egg Guide

An unofficial, community-driven Android companion app for **Call of Duty Zombies**.

The goal of this project is to bring Easter egg guides, map information, character biographies, storyline references, and other useful Zombies resources together in one convenient mobile application.

> [!IMPORTANT]
> **The Ultimate Easter Egg Guide is an unofficial fan project.**
>
> This project is not affiliated with, authorized by, sponsored by, or endorsed by Activision, Treyarch, Microsoft, or any other company associated with the Call of Duty franchise.

---

## About the Project

Call of Duty Zombies has accumulated years of maps, characters, storylines, Easter eggs, quests, and community discoveries. Information about these topics is often spread across wikis, videos, websites, forum posts, and community discussions.

The Ultimate Easter Egg Guide is being developed to organize that information into an accessible Android application.

Planned and developing features include:

* Main Easter egg walkthroughs
* Map selection and map-specific information
* Step-by-step quest instructions
* Character biographies
* Character groups and storyline information
* Zombies tools and reference material
* Images associated with maps, characters, and Easter egg steps
* Additional community resources

This application is currently under active development. Features, layouts, information, and available maps may change as development continues.

# Building the Project

## Requirements

* Android Studio
* Android SDK 36
* Java 11
* An Android device or emulator running API 24 or newer

## Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/Scorpio7736/The-Ultimate-Easter-Egg-Guide.git
   ```

2. Open the project in Android Studio.

3. Allow Gradle to synchronize and download the required dependencies.

4. Select an Android emulator or connected device.

5. Build and run the `app` configuration.
